Secure Local MCP Server Generator

Generate a custom MCP filesystem server sandboxed to one base folder

This generator creates a Node.js MCP server template that only permits filesystem access inside a configured base directory and supports stdio, HTTP, or both transports. It blocks path traversal, resolves symlinks conservatively, and rejects reads or writes outside the sandbox.

What the generated server does

Path sandboxing

Every requested path is resolved against the configured base directory, normalized, and verified to remain inside the base folder before any file operation is performed.

Traversal defense

Requests using sequences such as ../ are rejected after canonical path validation if they resolve outside the sandbox.

Controlled writes

Writes, overwrites, directory creation, and optional deletes only affect files beneath the allowed root.

Safe defaults

The server includes input validation, structured responses, conservative file handling, and clear setup instructions.

Run instructions

  1. Generate the server code below.
  2. Save it as server.js in a local project folder.
  3. Run npm init -y.
  4. Install the MCP SDK package shown in the generated header comments.
  5. Set the base folder environment variable or edit the constant in the file.
  6. Run the server locally with Node.js.
  7. Configure your MCP client to launch this local server.
